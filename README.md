### Dependencies

Main:
* Compiler with C++11 support: GCC >= 4.9.0 or Clang >= 3.5.1
* CMake >= 2.8.12
* Qt >= 5.6
* yaml-cpp >= 0.5.1
* Python >= 3.3
* Boost.Python >= 1.55.0
* dmidecode

Modules:
* welcome:
 * NetworkManager
 * UPower
* partition:
 * extra-cmake-modules
 * KF5: KCoreAddons, KConfig, KI18n, KIconThemes, KIO, KService
 * KPMcore >= 3.0
* bootloader:
 * systemd-boot or GRUB
* unpackfs:
 * squashfs-tools
 * rsync
